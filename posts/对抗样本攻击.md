## 白盒对抗样本攻击

白盒攻击假设攻击者可以完全访问模型的结构、参数以及梯度信息，是最强且最易实现的攻击方式。

对抗样本（Adversarial Examples）是指在原始输入数据中加入微小扰动后，能诱使深度神经网络产生错误分类的样本。

对于分类任务，模型学习目标：最小化损失函数
$$
\begin{aligned}
&\min_{\boldsymbol{\theta}} \ell\left(f_{\boldsymbol{\theta}}(\boldsymbol{x}), y\right) \\
&\boldsymbol{\theta}：模型参数（权重、偏置等） \\
&\boldsymbol{x}：输入样本（如图像） \\
&\boldsymbol{y}：真实标签 \\
&\ell(\cdot)：损失函数（如交叉熵）
\end{aligned}
$$

### 无目标攻击

无目标攻击的目标是在微小扰动下最大化分类损失，使模型对输入样本的预测与真实标签偏差最大：

$$
\begin{aligned}
&\max_{\boldsymbol{\delta}} \ell\left(f_{\boldsymbol{\theta}}(\boldsymbol{x} + \boldsymbol{\delta}), y\right)
\\
&\boldsymbol{\delta}：对抗扰动（噪声）
\end{aligned}
$$

为保证扰动人眼不可察，要限制扰动强度，每个维度扰动绝对值都小于 $\epsilon$

$$
\begin{aligned}
&\max_{\boldsymbol{\delta}} \ell\left(f_{\boldsymbol{\theta}}(\boldsymbol{x} + \boldsymbol{\delta}), y\right) \quad \text{s.t.} \quad \|\boldsymbol{\delta}\|_{\infty} \leq \epsilon 
\end{aligned}
$$

其中 $\epsilon$ 是最大允许扰动强度。这一过程本质上是梯度上升（Gradient Ascent），与模型训练时的梯度下降方向完全相反。

回顾梯度下降、反向传播的参数更新：
$$
\boldsymbol{\theta} = \boldsymbol{\theta} - \alpha \cdot \nabla_{\boldsymbol{\theta}} \ell\left(f_{\boldsymbol{\theta}}(\boldsymbol{x}), y\right) \\
$$
这个过程是最小化损失函数，如果反向更新，等同于在最大化损失函数
$$
\boldsymbol{\theta} = \boldsymbol{\theta} + \alpha \cdot \nabla_{\boldsymbol{\theta}} \ell\left(f_{\boldsymbol{\theta}}(\boldsymbol{x}), y\right)
$$
将更新做到输入上，即得对抗样本的公式：
$$
\tilde{\boldsymbol{x}} = \boldsymbol{x} + \alpha \cdot \nabla_{\boldsymbol{x}} \ell\left(f_{\boldsymbol{\theta}}(\boldsymbol{x}), y\right)
$$

#### FGSM

FGSM 是一种单步梯度攻击方法，通过在输入图像的梯度方向上添加一个固定大小的扰动来生成对抗样本。

实现 $ \|\boldsymbol{\delta}\|_{\infty} \leq \epsilon $ 的方法：

$$
\boldsymbol{\delta} = \epsilon \cdot \operatorname{sign}\left(\nabla_{\boldsymbol{x}} \ell\left(f_{\boldsymbol{\theta}}(\boldsymbol{x}), y\right)\right), 
\\
\operatorname{sign}(\delta_i) = \begin{cases} 
+1, & \delta_i > 0 \\
-1, & \delta_i \leq 0 
\end{cases}
$$

最终 FGSM 对抗样本生成流程

$$
\tilde{\boldsymbol{x}} = \boldsymbol{x} + \epsilon \cdot \operatorname{sign}\left(\nabla_{\boldsymbol{x}} \ell\left(f_{\boldsymbol{\theta}}(\boldsymbol{x}), y\right)\right)
$$

FGSM 攻击函数：

```python
def fgsm(imgs, epsilon, model, criterion, labels):
    model.eval()

    adv_xs = imgs.float()
    adv_xs.requires_grad = True

    # 模型前向传播，计算loss，然后loss反传
    o = model(adv_xs)
    loss = criterion(o, labels)
    model.zero_grad()
    loss.backward()

    # 得到输入的梯度、生成对抗样本
    # 公式：adv_xs = imgs + epsilon * sign(grad)，其中grad是输入的梯度，sign()是符号函数

    adv_xs = adv_xs + epsilon * adv_xs.grad.sign()

    # 对扰动做截断，保证对抗样本的像素值在合理域内
    # 使用函数：torch.clamp(input, min, max)对输入张量的元素进行截断，使其值在指定的范围内
    adv_xs = torch.clamp(adv_xs, 0.0, 1.0)

    model.train()

    return adv_xs.detach()
```

#### PGD

PGD 是 FGSM 的多步迭代版本，每次迭代后对扰动进行投影（clip）回 $\epsilon$ -ball 内，攻击能力更强。

Project Gradient Descent：每次 SGD 后都对扰动做裁剪，初始化扰动为 0，clamp 表示裁剪到 $[-\epsilon ,\epsilon]$ 中

$$
\delta_{t+1}=\operatorname{clamp}\left(\delta_{t}+\alpha \cdot \operatorname{sign}\left(\nabla_{x+\delta_{t}} \ell\left(f_{\theta}\left(x+\delta_{t}\right), y\right)\right),-\epsilon, \epsilon\right)
$$

代码实现

```python
def pgd(imgs, epsilon, alpha, iter, model, criterion, labels):

    model.eval()
    adv_xs = imgs.float()

    for i in range(iter):
        adv_xs = adv_xs.clone().detach().requires_grad_(True)
        # Forward and compute loss, then backward
        o = model(adv_xs)
        loss = criterion(o, labels)
        model.zero_grad()
        loss.backward()

        # Retrieve grad and generate adversarial example, note to detach

        adv_xs = adv_xs + alpha * adv_xs.grad.sign()

        # Clip perturbation
        adv_xs = torch.clamp(adv_xs - imgs, - epsilon, epsilon) + imgs
        adv_xs = torch.clamp(adv_xs, 0.0, 1.0)

    model.train()

    return adv_xs.detach()
```

### 有目标攻击

有目标攻击的目标是将模型预测引导到指定目标标签 $\tilde{y}$，即最小化预测值与目标标签的损失：

- 无目标攻击： $\max_{\boldsymbol{\delta}} \ell\left(f_{\boldsymbol{\theta}}(\boldsymbol{x} + \boldsymbol{\delta}), y\right)$ 在微小扰动下最大化损失函数
- 有目标攻击：在微小扰动下最小化到目标的距离 $\min_{\boldsymbol{\delta}} \ell\left(f_{\boldsymbol{\theta}}(\boldsymbol{x} + \boldsymbol{\delta}), \tilde{y}\right)$ 

#### FGSM

反向传播的目的是最小化预测值和真实值的 loss

而有目标攻击是最小化预测值和目标值的 loss

$$
\tilde{x}=x-\epsilon \cdot \operatorname{sign}\left(\nabla_{x} \ell\left(f_{\theta}(x), \tilde{y}\right)\right)
$$

其实只需要在无目标攻击的基础上对梯度符号取反：

```python
def fgsm_target(imgs, epsilon, model, criterion, labels):
    model.eval()

    adv_xs = imgs.float()
    adv_xs.requires_grad = True

    o = model(adv_xs)
    loss = criterion(o, labels)
    model.zero_grad()
    loss.backward()

    adv_xs = adv_xs - epsilon * adv_xs.grad.sign()

    adv_xs = torch.clamp(adv_xs, 0.0, 1.0)

    model.train()

    return adv_xs.detach()
```

$$

## 黑盒对抗样本攻击

黑盒攻击假设攻击者无法访问模型内部梯度，只能通过查询模型输出（logits 或预测概率）来估计梯度，模拟真实 API 调用场景。

### 单步 FGSM NES

NES（Natural Evolution Strategies）利用高斯噪声采样估计梯度方向。

```python
def nes(imgs, epsilon, model, labels, sigma, n):
    model.eval()

    adv_xs = imgs.reshape(-1, 28*28).float().clone().detach()  # 加 clone 更安全
    grad = torch.zeros_like(adv_xs)   # 用于累加梯度估计
  
    # 通过多次噪声采样来估计梯度
    for _ in range(n):
        u = torch.randn_like(adv_xs)  # 生成标准正态分布噪声 u ~ N(0, I)
      
        # 在原始对抗样本的两侧添加噪声（正向和负向采样）
        x_plus = torch.clamp(adv_xs + sigma * u, 0.0, 1.0)
        x_minus = torch.clamp(adv_xs - sigma * u, 0.0, 1.0)

        # 恢复为卷积网络要求的 4D 张量格式 [batch, 1, 28, 28]
        x_plus_img = x_plus.reshape(-1, 1, 28, 28)
        x_minus_img = x_minus.reshape(-1, 1, 28, 28)

        # 使用 reduction='none' 得到每个样本独立的 loss（不能用默认 mean）
        loss_fn = nn.CrossEntropyLoss(reduction='none')
        loss_plus = loss_fn(model(x_plus_img), labels)      # shape: [batch]
        loss_minus = loss_fn(model(x_minus_img), labels)    # shape: [batch]

        # NES 梯度估计核心公式：(loss_plus - loss_minus) / (2 * sigma) * u
        # 这是一种无偏的梯度估计方法
        grad += (loss_plus - loss_minus).unsqueeze(1) * u / (2 * sigma)

    # 平均梯度
    grad = grad / n

    # 单步 FGSM 更新：沿估计梯度的符号方向添加扰动
    adv_xs = adv_xs + epsilon * grad.sign()
    adv_xs = torch.clamp(adv_xs, 0.0, 1.0)

    adv_xs = adv_xs.view_as(imgs)      # 恢复 [batch, 1, 28, 28]

    model.train()
    return adv_xs.detach()

```

### 多步 FGSM 迭代 NES

在单步基础上加入迭代投影

```python
def nes(imgs, epsilon, model, labels, sigma, n):
    model.eval()

    # 保存原始图像用于后续投影
    original = imgs.reshape(-1, 28 * 28).float().clone().detach()
    adv_xs = original.clone()   # 初始化对抗样本

    # 多步迭代参数
    num_steps = 20      # 迭代次数
    alpha = 0.02        # 每步步长

    for step in range(num_steps):
        grad = torch.zeros_like(adv_xs)

        # NES 梯度估计（与单步版本相同）
        for i in range(n):
            u = torch.randn_like(adv_xs)                     # u ~ N(0, I)

            x_plus = adv_xs + sigma * u
            x_minus = adv_xs - sigma * u

            x_plus_img = x_plus.reshape(-1, 1, 28, 28)
            x_minus_img = x_minus.reshape(-1, 1, 28, 28)

            loss_fn = nn.CrossEntropyLoss(reduction='none')
            loss_plus = loss_fn(model(x_plus_img), labels)
            loss_minus = loss_fn(model(x_minus_img), labels)
          
            grad_i = (loss_plus - loss_minus) / (2 * sigma)
            grad += grad_i.unsqueeze(1) * u

        grad = grad / n
      
        # 沿梯度方向更新（类似 PGD 的迭代）
        adv_xs = adv_xs + alpha * grad.sign()

        # 重要：投影回 epsilon-ball（防止扰动过大）
        delta = torch.clamp(adv_xs - original, -epsilon, epsilon)
        adv_xs = torch.clamp(original + delta, 0.0, 1.0)
      
	# 恢复图像形状
    adv_xs = adv_xs.view_as(imgs)
    model.train()
    return adv_xs.detach()
```
