RobotsеҚҸи®®

RobotsеҚҸи®®пјҲд№ҹз§°дёәзҲ¬иҷ«еҚҸи®®жҲ–жңәеҷЁдәәеҚҸи®®пјүжҳҜдёҖз§Қз”ЁдәҺжҢҮеҜјжҗңзҙўеј•ж“ҺзҲ¬иҷ«иЎҢдёәзҡ„зәҰе®ҡгҖӮе®ғйҖҡиҝҮеңЁзҪ‘з«ҷж №зӣ®еҪ•дёӢж”ҫзҪ®дёҖдёӘеҗҚдёәrobots.txtзҡ„ж–Үжң¬ж–Үд»¶пјҢе‘ҠзҹҘзҲ¬иҷ«е“ӘдәӣйЎөйқўеҸҜд»ҘжҠ“еҸ–пјҢе“ӘдәӣйЎөйқўеә”иў«зҰҒжӯўжҠ“еҸ–пјҢд»ҺиҖҢдҝқжҠӨзҪ‘з«ҷж•°жҚ®йҡҗз§ҒгҖҒдјҳеҢ–жңҚеҠЎеҷЁжҖ§иғҪе№¶йҒҝе…ҚиҝҮеәҰжҠ“еҸ–гҖӮ

phpж–Үд»¶еҢ…еҗ«жјҸжҙһ

дёәдәҶжӣҙеҘҪең°дҪҝз”Ёд»Јз Ғзҡ„йҮҚз”ЁжҖ§пјҢеҸҜд»ҘдҪҝз”Ёж–Үд»¶еҢ…еҗ«еҮҪж•°е°Ҷж–Үд»¶еҢ…еҗ«иҝӣжқҘпјҢзӣҙжҺҘдҪҝз”Ёж–Үд»¶дёӯзҡ„д»Јз ҒжқҘжҸҗй«ҳйҮҚз”ЁжҖ§гҖӮ

дҪҶжҳҜиҝҷд№ҹдә§з”ҹдәҶж–Үд»¶еҢ…еҗ«жјҸжҙһпјҢдә§з”ҹеҺҹеӣ жҳҜеңЁйҖҡиҝҮ PHP зҡ„еҮҪж•°еј•е…Ҙж–Үд»¶ж—¶пјҢдёәдәҶзҒөжҙ»еҢ…еҗ«ж–Үд»¶дјҡе°Ҷиў«еҢ…еҗ«ж–Үд»¶и®ҫзҪ®дёәеҸҳйҮҸпјҢйҖҡиҝҮеҠЁжҖҒеҸҳйҮҸжқҘеј•е…ҘйңҖиҰҒеҢ…еҗ«зҡ„ж–Үд»¶гҖӮжӯӨж—¶з”ЁжҲ·еҸҜд»ҘеҜ№еҸҳйҮҸзҡ„еҖјеҸҜжҺ§пјҢиҖҢжңҚеҠЎеҷЁз«ҜжңӘеҜ№еҸҳйҮҸеҖјиҝӣиЎҢеҗҲзҗҶең°ж ЎйӘҢжҲ–иҖ…ж ЎйӘҢиў«з»•иҝҮпјҢе°ұдјҡеҜјиҮҙж–Үд»¶еҢ…еҗ«жјҸжҙһгҖӮ

ж–Үд»¶еҢ…еҗ«еҮҪж•°

еҠҹиғҪ еҮҪж•°
д»Јз Ғжү§иЎҢеҲ° include() еҮҪж•°ж—¶е°Ҷж–Үд»¶еҢ…еҗ« include()
еҪ“йҮҚеӨҚи°ғз”ЁеҗҢдёҖж–Үд»¶ж—¶еҸӘи°ғз”ЁдёҖж¬Ў,еҠҹиғҪдёҺ include() зӣёеҗҢ include_once()
require() жү§иЎҢеҰӮжһңеҸ‘з”ҹй”ҷиҜҜпјҢеҮҪж•°дјҡжҠҘй”ҷе№¶з»Ҳжӯўи„ҡжң¬ require()
еҪ“йҮҚеӨҚи°ғз”ЁеҗҢдёҖж–Үд»¶ж—¶еҸӘи°ғз”ЁдёҖж¬Ў,еҠҹиғҪдёҺ require() зӣёеҗҢ require_once()

phpзү№жҖ§

PHPеәҸеҲ—еҢ–дёҺеҸҚеәҸеҲ—еҢ–

  • PHPеҶ…зҪ®еәҸеҲ—еҢ–еҮҪж•° serialize()пјҢз”ЁдәҺжҠҠPHPзҡ„ж•°жҚ®з»“жһ„пјҲеҰӮеҜ№иұЎгҖҒж•°з»„пјүиҪ¬жҚўжҲҗдёҖдёӘеӯ—з¬ҰдёІиЎЁзӨәпјҢдҫҝдәҺеӯҳеӮЁжҲ–дј иҫ“гҖӮ

  • еҜ№еә”еҸҚеәҸеҲ—еҢ–еҮҪж•°unserialize()пјҢдҪңз”ЁжҳҜпјҡе°ҶеәҸеҲ—еҢ–еҗҺзҡ„еӯ—з¬ҰдёІиҝҳеҺҹдёәPHPзҡ„ж•°жҚ®з»“жһ„пјҲжҜ”еҰӮеҜ№иұЎгҖҒж•°з»„зӯүпјүгҖӮ

unserialize() еңЁи°ғз”ЁеүҚдјҡжЈҖжҹҘжҳҜеҗҰеӯҳеңЁдёҖдёӘ __wakeupж–№жі•гҖӮеҰӮжһңеӯҳеңЁпјҢеҲҷдјҡе…Ҳи°ғз”Ё __wakeup ж–№жі•пјҢйў„е…ҲеҮҶеӨҮеҜ№иұЎж•°жҚ®гҖӮ

PHP еҸҚеәҸеҲ—еҢ–жјҸжҙһ / зү№жҖ§ вҖ”вҖ” еҪ“еҜ№иұЎеұһжҖ§ж•°йҮҸеЈ°жҳҺдёҺе®һйҷ…дёҚз¬Ұж—¶пјҢ__wakeup()дёҚдјҡиў«и°ғз”Ё

CVE-2016-7124

еәҸеҲ—еҢ–ж јејҸ

  • String : s:size:value;
  • Integer : i:value;
  • Boolean : b:value;(дҝқеӯҳ1жҲ–0)
  • Null : N;
  • Array : a:size:{key definition;value definition;(repeated per element)}
  • Object : O:strlen(object name):object name:object size:{s:strlen(property name):property name:property definition;(repeated per property)}

phpдјӘеҚҸи®®

еңЁ PHP дёӯпјҢ дјӘеҚҸи®®пјҲPseudo Protocolsпјү д№ҹиў«з§°дёә жөҒеҢ…иЈ…еҷЁ пјҢиҝҷдәӣдјӘеҚҸи®®д»Ҙ php:// ејҖеӨҙпјҢеҗҺйқўи·ҹзқҖдёҖдәӣеҸӮж•°пјҢз”ЁдәҺжҢҮе®ҡ иҰҒжү§иЎҢзҡ„ж“ҚдҪң жҲ– йңҖиҰҒи®ҝй—®

дјӘеҚҸи®®иЎЁжҳҺиҝҷдәӣеҚҸи®®е№¶дёҚжҳҜдёҖдёӘ зңҹе®һзҡ„еӨ–йғЁеҚҸи®® пјҢдҫӢеҰӮ http жҲ– ftp гҖӮ

PHP дјӘеҚҸи®®зҡ„еҮәзҺ°жҳҜдёәдәҶжҸҗдҫӣдёҖдёӘ з»ҹдёҖзҡ„ гҖҒ з®ҖжҙҒзҡ„ жҺҘеҸЈжқҘеӨ„зҗҶ дёҚеҗҢзҡ„ж•°жҚ®жөҒ дјӘеҚҸи®®еҸҜд»Ҙиў«зңӢдҪңжҳҜдёҖз§ҚжЎҘжўҒ пјҢе®ғд»¬е…Ғи®ёејҖеҸ‘иҖ… дҪҝз”Ёеёёи§„зҡ„ж–Үд»¶ж“ҚдҪңеҮҪж•°жқҘеӨ„зҗҶеҗ„з§ҚдёҚеҗҢзҡ„ж•°жҚ®жөҒ гҖӮ

еҮ дёӘ PHP ж”ҜжҢҒзҡ„дјӘеҚҸи®®еҰӮдёӢпјҡ

дјӘеҚҸи®® еҠҹиғҪ
file:// и®ҝй—®жң¬ең°ж–Үд»¶зі»з»ҹ
http:// и®ҝй—® HTTP(s) зҪ‘еқҖ
php:// и®ҝй—®еҗ„дёӘиҫ“е…Ҙ/иҫ“еҮәжөҒ
phar:// PHP еҪ’жЎЈ
zip:// еҺӢзј©жөҒ

php://filter дјӘеҚҸи®®пјҡ

php://filterдё»иҰҒз”ЁдәҺеңЁиҜ»еҸ–ж–Үд»¶ж—¶иҝӣиЎҢж•°жҚ®жөҒиҝҮж»ӨпјҲеҰӮBase64зј–з ҒгҖҒROT13иҪ¬жҚўзӯүпјүгҖӮе®ғзҡ„еёёи§Ғз”ЁйҖ”жҳҜ иҜ»еҸ–PHPж–Үд»¶жәҗз ҒпјҲеӣ дёәзӣҙжҺҘ includeжҲ– file_get_contentsдјҡжү§иЎҢPHPд»Јз ҒпјҢиҖҢиҝҮж»ӨеҗҺеҸҜд»ҘиҺ·еҸ–еҺҹе§ӢеҶ…е®№пјүгҖӮ

дёҖдәӣиҝҮж»ӨеҷЁпјҡ

  1. зј–з ҒиҪ¬жҚўиҝҮж»ӨеҷЁ (convert.*)

  1. еӯ—з¬ҰдёІиҝҮж»ӨеҷЁ (string.*)

  2. еҺӢзј©иҝҮж»ӨеҷЁ (zlib.*, bzip2.*)
